== Installation ==

1. Upload the `awesome-coming-soon` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Settings' > 'Awesome Coming Soon' to configure your coming soon page

== Frequently Asked Questions ==

= Can I use my own logo and branding? =

Yes, Awesome Coming Soon allows you to easily upload your own logo and customize colors to match your brand.

= Will search engines be able to index my real site while the coming soon page is active? =

By default, Awesome Coming Soon sends the appropriate meta tags to prevent search engine indexing. You can customize these settings in the SEO options.

= Can I use this for a quick maintenance mode? =

Yes, Awesome Coming Soon is perfect for both extended coming soon pages and quick maintenance mode scenarios.
